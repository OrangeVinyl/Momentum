## 🚀 구현 기능

### ✅ 유저 닉네임 저장
처음으로 들어온 화면에서 닉네임 작성 시, 로컬 스토리지에 저장됨
### ✅ ToDoList 작성 및 삭제
Input에 일정을 입력 시, 하단의 리스트에 반영되고 버튼을 클릭하면 지워지는 기능 탑재
### ✅ 날씨 API 활용하여 화면 표시
날씨 API를 받아와 위치 기반의 서비스 제공
### ✅ 랜덤으로 나오는 명언
페이지를 새로고침 할때마다 명언이 랜덤하게 표기됨
### ✅ 랜덤으로 나오는 화면
페이지를 새로고침 할때마다 화면이 랜덤하게 나타남
